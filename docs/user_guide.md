# Panduan Pengguna Virtual Photosynthesis Lab

## 1. Tentang Aplikasi
Aplikasi ini digunakan untuk pembelajaran fotosintesis secara virtual.

## 2. Parameter yang Bisa Diatur
- Intensitas cahaya (0–2000)
- Konsentrasi CO₂
- Suhu lingkungan

## 3. Output
- Grafik kurva respon cahaya
- Produksi oksigen (mg/L/jam)
- File CSV hasil simulasi

## 4. Tujuan Pembelajaran
- Memahami pengaruh cahaya, CO₂, dan suhu terhadap fotosintesis
- Melatih analisis data fisiologi tanaman
